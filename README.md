A lightweight and responsive To-Do application built using HTML, CSS, and Vanilla JavaScript.
The app allows users to manage daily tasks efficiently with persistent storage using the browser’s LocalStorage API.

🛠 Tech Stack
HTML5
CSS3
JavaScript (ES6)
LocalStorage API

🚀 Features
Add new tasks
Delete tasks
Persistent storage using LocalStorage
Responsive UI
Clean and minimal design
Instant DOM updates without page reload

🧠 Key Concepts Implemented
DOM Manipulation
Event Handling
State Management
Local Storage Persistence
Dynamic Rendering
CRUD Operations
